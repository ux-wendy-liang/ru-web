# Wendy 作品集网站迁移计划

## 📋 项目概述

将你现有的 Webflow 作品集网站（https://wendyliang.webflow.io/）迁移到本地开发环境，让你拥有完全的控制权和自定义能力。

---

## 🎨 你现有网站的特点

根据我的分析，你的网站包含以下内容：

### 页面结构
- **首页**：展示你的简介和 4 个主要项目
- **关于页面**：详细介绍你的背景（8+ 年经验，LinkedIn、腾讯等）
- **项目详情页**：每个项目的详细案例研究

### 设计风格
- 主色调：清新的绿色系（#F2FCE2）
- 字体：Lato 字体家族，简洁现代
- 布局：卡片式设计，每个项目有缩略图
- 响应式设计：在手机、平板、电脑上都能正常显示

### 主要项目
1. LinkedIn Publisher Portal（2024）
2. Knowledge Graph Management Tool（2023）
3. LinkedIn Data Virtualization Dashboard（2022）
4. Camera App（2014）

---

## 🛠️ 技术选型（不用担心，我会帮你设置好）

- **React**：用来构建网站界面的工具（就像 Figma 组件一样，但是是代码版）
- **TypeScript**：让代码更安全可靠
- **Vite**：快速开发工具，修改后立即看到效果（热重载）
- **Tailwind CSS**：快速实现你的设计样式
- **React Router**：实现页面之间的跳转

## 📂 项目文件结构（类比设计文件组织）

```
ru-web/                          # 项目根目录（像 Figma 文件）
├── src/                         # 源代码文件夹
│   ├── components/              # 可复用组件（像 Figma Components）
│   │   ├── Navigation.tsx       # 导航栏
│   │   ├── Footer.tsx           # 页脚
│   │   ├── ProjectCard.tsx      # 项目卡片
│   │   └── Button.tsx           # 按钮组件
│   ├── pages/                   # 页面（像 Figma Pages）
│   │   ├── Home.tsx             # 首页
│   │   ├── About.tsx            # 关于页
│   │   └── ProjectDetail.tsx    # 项目详情页模板
│   ├── assets/                  # 资源文件
│   │   ├── images/              # 图片
│   │   │   ├── projects/        # 项目图片
│   │   │   └── profile/         # 个人照片
│   │   └── fonts/               # 字体文件（如需要）
│   ├── data/                    # 数据文件
│   │   └── projects.ts          # 项目信息数据
│   ├── styles/                  # 样式文件
│   │   └── globals.css          # 全局样式
│   └── App.tsx                  # 应用主文件
├── public/                      # 公共资源
└── package.json                 # 项目配置文件
```
